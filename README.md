# Grow ZSH Complete
A ohmyzsh compatible plugin for [grow](https://grow.io) that adds auto-complete

## Getting Started
For use with oh-my-zsh
```bash
git clone https://github.com/drGrove/grow-completion ~/.oh-my-zsh/custom/plugins/grow
```

```
# ~/.zshrc

plugins=(grow)
```

```bash
source ~/.zshrc
```
